# ☕ Estudos de Java - Programação Orientada a Objetos

Este repositório contém exercícios e pequenos projetos desenvolvidos durante meus estudos de **Programação Orientada a Objetos em Java**.

O objetivo é praticar conceitos fundamentais da orientação a objetos e melhorar minha lógica de programação.

## 🚀 Conceitos praticados

* Classes
* Objetos
* Atributos
* Métodos
* Encapsulamento
* Organização em pacotes
* Reutilização de código

## 📂 Exercícios

### 1️⃣ Cálculo da área de dois triângulos

Projeto que compara a área de dois triângulos utilizando a **Fórmula de Heron**.

Neste exercício foram aplicados:

* Criação de classes
* Criação de métodos
* Instanciação de objetos
* Separação de responsabilidades
