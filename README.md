# Estudos de Java - Programação Orientada a Objetos

Este repositório contém exercícios e pequenos projetos desenvolvidos durante meus estudos de **Programação Orientada a Objetos em Java**.

O objetivo é praticar conceitos fundamentais de POO e melhorar minha lógica de programação.

## Conceitos praticados

- Classes
- Objetos
- Atributos
- Métodos
- Encapsulamento
- Organização em pacotes
- Reutilização de código

## Exercícios

### Área de Triângulos
Programa que compara a área de dois triângulos utilizando a Fórmula de Heron.

Estrutura:

entities/
 └── Triangle.java

application/
 └── Program.java

---

### Controle de Estoque de Produtos
Programa simples que simula o gerenciamento de estoque de um produto.

Funcionalidades:
- Adicionar produtos ao estoque
- Remover produtos do estoque
- Calcular valor total em estoque

Estrutura:

entities_2/
 └── Product.java

application_2/
 └── Program2.java

## Tecnologias utilizadas

- Java
- IntelliJ IDEA

## Objetivo

Este repositório registra minha evolução prática em Java e Programação Orientada a Objetos.
